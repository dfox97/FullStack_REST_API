Instructions

The Web server is developed using REST API with PHP and MYSQL. The server demonstrates a basic CRUD REST application consisting of Read (GET), Write(POST), Remove(DELETE),Change (UPDATE).


  • Retrieve information on all the teams (this does not include information on the players of each team, but should include a link to the collection resource for all players of each team), sorted by team names;
    • Link: https://student.csc.liv.ac.uk/~sgdfox/v1/read
    • Method:GET
    • Result:Shows all teams

  • Retrieve information on all players of a specific team;
    • Link: https://student.csc.liv.ac.uk/~sgdfox/v1/Players/tid/{tid:num}
    • Example:https://student.csc.liv.ac.uk/~sgdfox/v1/players/tid/1
    • Method:GET
    • Result: {"pid":1,"tid":1,"surname":"Jones","firstname":"John","nationality":"British","birth_date":"1995-12-12"},{"pid":2,"tid":1,...}

  • Retrieve information on an existing player of a team;
    • Link: student.csc.liv.ac.uk/~sgdfox/v1/Players/tid/{tidnum}/pid/{pidnum}
    • Example:https://student.csc.liv.ac.uk/~sgdfox/v1/players/tid/1/pid/2
    • Method:GET
    • Result: {"pid":2,"tid":1,"surname":"Bob","firstname":"Lazar","nationality":"French","birth_date":"1997-01-10"}

  • Add a player / team member to an existing team;
    • Link: student.csc.liv.ac.uk/~sgdfox/v1/Players/tid/{tidnum}
    • Example:https://student.csc.liv.ac.uk/~sgdfox/v1/players/tid/1
    • Method:POST
    • JSON input:
      {
        "pid":"4",
        "surname":"Fox",
        "firstname":"Daniel",
        "nationality":"british",
        "birth_date":"1997-01-23"
      }
    • Result:Will be inserted in mysql, tid is set by the link url {tidnum}.This can be verified by calling the GET method to view

  • Delete an existing player / team member from a team;
    • Link: student.csc.liv.ac.uk/~sgdfox/v1/Players/tid/{tidnum}/pid/{pidnum}
    • Example:https://student.csc.liv.ac.uk/~sgdfox/v1/players/tid/2/pid/5
    • Method:DELETE
    • JSON input not required as player is removed from primary key pid number.
    • Result:Will be deleted in mysql, tid and pid is set by the link url {tidnum} and {pidnum} .This can be verified by calling the GET method to view
  • update information for an existing player / team member of a team.
    • Link: student.csc.liv.ac.uk/~sgdfox/v1/Players
    • Example:https://student.csc.liv.ac.uk/~sgdfox/v1/Players
    • Method:PUT
    • JSON input:
      {
        "pid":"4",
        "surname":"Fox",
        "firstname":"Daniel",
        "nationality":"British",
        "birth_date":"1997-01-23"
        "tid":"3"
      }
    • Result:Will be deleted in mysql, tid and pid is set by the link url {tidnum} and {pidnum} .This can be verified by calling the GET method to view
